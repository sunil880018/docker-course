If you delete node_modules from the local machine ,container will crash because container dont know nodemon
So run this command------------


1. docker run -v %cd%:/app -v /app/node_modules  -p 3000:3000 -d --name <container_name> <image_name>

//now your container run successfully,but this is used only development purpose


# read only mode <ro> because we dont want to create any file inside the container
2.docker run -v %cd%:/app:ro -v /app/node_modules  -p 3000:3000 -d --name <container_name> <image_name>
